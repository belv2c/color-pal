What are HTML color codes?

Color codes represent the colors red, green and blue in a format that computers are able to interpret and display.
Commonly used formats are Hex color codes, RGB, HSL, and HTML color names.

Hex Color Codes

A hexidecimal is a six digit numeric representation of a color which is split into three parts. Hex values are composed of numbers from 0 to 9 and letters from A to F.

# xx xx xx

0 is the lowest value in a hex code - it represents the absence of color.

#000000

F is the highest value in a hex code - it represents the highest amount of brightness

#FFFFFF

With a basic understanding of how hex codes work, it is also easy to create gray colors, which consist of an equal value of red, green and blue.

#808080
#D3D3D3
#F5F5F5

Abbreviated Hex Colors

With the possibility of 16 million different color combinations, using hex values can become overwhelming. You can simplify the possibility of combinations to around 4,000 by abbreviating hex codes with one digit for red, one for blue and one for green.

# x x x

But why would I even use hex abbreviations?

The use of shorthand hex colors can help assist with optimizing your style sheets to their minimum size.

Here are some examples of shorthand hex notation:

#ff6600
#f60
#ffcc00
#fc0
